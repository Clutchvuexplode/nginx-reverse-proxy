#### Шаг 2: Временно отключите антивирус
- **Важно:** Антивирус может пометить инструмент (ложное срабатывание)

#### Шаг 3: Настройка и запуск
- Нажмите правой кнопкой мыши `nginx_reverse_proxy.exe`

- Отредактируйте `nginx.conf` для настройки правил прокси
- Укажите адреса бэкенд-серверов
- Настройте SSL-сертификаты (опционально)
- Выполните `nginx -t` для проверки конфигурации
- Выполните `nginx` для запуска прокси-сервера
